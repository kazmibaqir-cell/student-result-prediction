# Student Result Prediction

A Machine Learning project that predicts student performance based on study hours and attendance.

## Features

* Predict student results using machine learning
* Data preprocessing and analysis
* Model training and evaluation
* Easy to understand project structure

## Technologies Used

* Python
* NumPy
* Pandas
* Scikit-Learn
* Matplotlib

## Project Structure

student-result-prediction/

├── data.csv

├── train.py

├── requirements.txt

└── README.md

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python train.py
```
